## Selection Criteria

1. **Search volume** — 500-5,000 monthly searches (sweet spot)
2. **Competition** — No dominant calculator in top 3 results
3. **User intent** — "X calculator" queries (not "how does X work")
4. **Monetization** — Finance = higher CPMs, tool-adjacent = affiliate potential
5. **Build complexity** — Start simple, add depth later
