## Calculator Selection Criteria

When prioritizing which calculator to build next, consider:

### 1. Search Volume
- Target: 500-5,000 monthly searches
- Check: Google Keyword Planner, Ahrefs, Ubersuggest
- Sweet spot: Enough traffic, low competition

### 2. Competition
- Look for: No dominant calculators in top 3 results
- Avoid: Keywords dominated by major finance sites
- Good sign: Generic or outdated tools ranking

### 3. User Intent
- High intent: "freelance tax calculator" (wants to calculate)
- Low intent: "how do freelance taxes work" (wants info)
- Target: Calculator-specific queries

### 4. Monetization Potential
- Finance calculators → Higher CPM
- B2B calculators → Lead gen potential
- Tool-adjacent → Affiliate opportunities

### 5. Build Complexity
- Simple: Basic math, few inputs (1-2 hours)
- Medium: Multiple scenarios, conditionals (2-4 hours)
- Complex: External data, many edge cases (4+ hours)
- Start simple, add complexity later
